Menu Data Extraction from Image

We created a web page that allows users to upload an unstructured restaurant menu, which is then organized and returned in structured JSON format. The uploaded Menu is processed using a clustering algorithm. The app can process up to ten menus concurrently.

Webpage Creation from Image

We developed a web application that enables users to upload a web design, which will be processed and identifies the position of the basic elements and returns an HTML file of the design uploaded.
